# 故事转视频脚本 - 通用转换工具

## 使用指南

本工具帮助您将任何故事（小说、短篇、真实事件）快速转换为符合 Seedance 2.0 AI 视频制作标准的专业分镜头剧本。

**重要：本工具生成的剧本格式严格遵循"好剧本.md"质量标准。**

---

## 第一步：核心梗与故事分析

### 1.1 核心梗定位

在开始转换前，先确定故事的核心梗（2-4字概括）：

| 核心梗类型 | 示例 | 说明 |
|-----------|------|------|
| 绝境反杀 | 林冲风雪山神庙 | 主角被逼入绝境后爆发反击 |
| 复仇爽剧 | 基督山伯爵 | 遭受背叛后展开复仇 |
| 治愈温馨 | 多拉A梦 | 温暖人心的日常故事 |
| 悬疑惊悚 | 命运石之门 | 充满悬念的剧情 |
| 成长励志 | 灌篮高手 | 主角通过努力成长 |
| 爱情故事 | 泰坦尼克号 | 感情线为主轴 |

### 1.2 故事梗概六要素

填写此表来构建完整故事框架：

| 要素 | 填写内容 | 示例（林冲） |
|------|----------|-------------|
| **故事背景** | 时间、地点、时代背景 | 北宋末年，沧州牢城营，风雪交加的寒冬 |
| **开场冲突** | 什么事件打破现状 | 林冲遭陷害发配，奉命看守草料场，不知杀机已至 |
| **主角画像** | 谁是主角，核心特质 | 武艺高强却一味隐忍，只想服刑归乡，处于崩溃边缘 |
| **主线事件** | 核心情节推进 | 草料场雪压塌 → 山神庙暂避 → 听到阴谋 → 绝地反击 |
| **结局** | 最终结果和转变 | 手刃仇敌，斩断归乡幻想，雪夜上梁山 |

### 1.3 一句话卖点

**公式：** [情绪号召] + [核心冲突] + [视觉高潮画面]

**示例：**
- 林冲：忍无可忍，无须再忍！风雪夜，豹子头林冲长枪出鞘，血溅山神庙！
- 司马光：临危不乱，机智救人！孩童司马光砸缸救人，展现超乎常人的冷静！

## 第二步：人物小传构建

每个主要角色都需要完整的小传：

### 人物小传模板

```
角色名
视觉形象：[年龄] + [外貌特征] + [服装细节] + [标志性动作/道具]
身份背景：[社会身份] + [与主角关系] + [过往经历]
核心标签：[标签1] + [标签2] + [标签3-4个]
性格特点：[正常状态] → [压力下表现] → [爆发/转折时]
金句：[最能代表角色的台词]
```

### 示例对比

**❌ 简单版（不可用）：**
```
林冲：30多岁，禁军教头，性格隐忍
```

**✅ 完整版（标准格式）：**
```
林冲
视觉形象：三十多岁，面容刚毅沧桑，身穿破旧棉袄，头戴毡笠，手持花枪，腰悬酒葫芦。
身份背景：前八十万禁军教头，现沧州囚犯。
核心标签：隐忍不发、悲情英雄、杀神降临。
性格特点：原本委曲求全，安分守己，被逼入绝境后展现出极强的爆发力与决绝。
金句："这世道，逼好人做鬼！"
```

## 第三步：剧本大纲（起承转合）

### 三幕式 / 四幕式结构

根据时长选择结构：

| 时长 | 结构 | 每幕内容 |
|------|------|----------|
| 5集 × 15秒 | 简化三幕 | 起(1集) → 承/转(2-3集) → 高潮/合(4-5集) |
| 10集 × 15秒 | 标准三幕 | 起(1-2集) → 承(3-6集) → 转/合(7-10集) |
| 20集 × 15秒 | 四幕结构 | 起(1-5集) → 承(6-10集) → 转(11-15集) → 合(16-20集) |

### 大纲模板

```
前期（起）：[场景建立] + [主角现状] + [触发事件]
中期（承/转）：[冲突发展] + [2-3次挫折] + [转折点]
后期（高潮）：[最终对决] + [情感/动作爆发]
尾声（收尾）：[结局状态] + [主题升华] + [余韵]
```

### 示例（林冲 - 5集版）

```
前期（起）：林冲独自看守草料场，环境恶劣，心境孤寂压抑。
中期（承/转）：草屋被雪压塌，林冲被迫前往山神庙暂避，意外撞破陆谦等人的火烧草料场阴谋。
后期（高潮）：林冲听闻真相，怒火攻心，冲出庙门，在风雪中格杀陆谦等人。
尾声（收尾）：林冲提枪远去，消失在茫茫风雪中，从此江湖路远。
```

## 第四步：剧本正文撰写（核心）

### 4.1 标准分镜头剧本格式

**每集必须遵循以下格式：**

```
第X集
X-X [日/夜] [内/外] [场景名称]
道具：[道具1]、[道具2]
出场人物：[角色1]、[角色2]

△ 【空镜】[开场氛围镜头，详细的画面、运镜、光影描述]
△ [镜头2 - 具体运镜 + 画面 + 动作]
△ [镜头3 - 继续推进情节]
角色名（os）：[内心独白/旁白]
△ [反应镜头 - 面部表情、肢体语言]
角色名：[对白]
△ [动作序列 - 用 → 连接连续动作]
【字幕：xxx】[需要字幕时添加]
△ 【闪回】[闪回场景]
[闪回镜头内容]
【闪回结束】
△ [回到现实 - 情绪反应]
角色名（vo）：[画外音，人物不在画面中]
△ [高潮动作镜头]
△ 【空镜】[结尾氛围镜头]
【字幕：xxx】[结尾字幕]
```

### 4.2 镜头描述速查表

#### 运镜术语

| 运镜 | 中文 | 用途 | 示例 |
|------|------|------|------|
| Push In | 推镜头 | 强调情绪，聚焦主体 | 推近至林冲面部特写 |
| Pull Out | 拉镜头 | 展现环境，营造孤独感 | 拉远展示茫茫雪原 |
| Pan | 摇镜头 | 展示场景全貌 | 从左摇至右，展现废墟 |
| Truck/Track | 移镜头 | 跟随横向移动 | 移镜头跟随奔跑 |
| Follow Shot | 跟镜头 | 跟随主体移动 | 跟拍林冲行走 |
| Orbit/Rotate | 环绕镜头 | 360度展示主体 | 环绕林冲旋转 |
| Crane | 升降镜头 | 俯冲或拉升 | 俯冲至地面 |
| Hitchcock Zoom | 希区柯克变焦 | 背景压缩，营造压迫感 | |
| One Shot | 一镜到底 | 不切换镜头 | |
| Handheld | 手持晃动 | 纪实感，紧张感 | |

#### 景别术语

| 景别 | 画面范围 | 用途 |
|------|----------|------|
| 远景 | 环境全景，人物很小 | 建立场景，展现氛围 |
| 全景 | 人物全身 | 展现动作，空间关系 |
| 中景 | 人物腰部以上 | 常用对话场景 |
| 近景 | 人物胸部以上 | 情感交流 |
| 特写 | 面部或细节 | 强调情绪，关键信息 |
| 大特写 | 眼睛/手部等细节 | 极致情绪，重要线索 |

### 4.3 对白标注格式

| 标注 | 含义 | 示例 |
|------|------|------|
| `角色名（os）` | 画外音/内心独白 | 林冲（os）：这世道... |
| `角色名（vo）` | 画外音，人物不在画面 | 陆谦（vo）：这把火下去... |
| `角色名（情绪）` | 带情绪的对白 | 林冲（怒吼）：奸贼！纳命来！ |
| `角色名` | 普通对白 | 林冲：且去那山神庙暂避一夜。 |

### 4.4 特殊元素标注

| 元素 | 用途 | 示例 |
|------|------|------|
| `【空镜】` | 氛围镜头，无人物 | △ 【空镜】水墨风格渲染的沧州城外 |
| `【闪回】` | 回忆镜头 | △ 【闪回】高衙内调戏妻子的画面... |
| `【闪回结束】` | 回到现实 | 【闪回结束】 |
| `【字幕】` | 屏幕文字 | 【字幕：沧州草料场 看守 林冲】 |

### 4.5 动作序列描述

**使用箭头 → 连续动作：**

```
❌ 错误：林冲打死了三个人

✅ 正确：
△ 林冲如天神下凡般冲出，花枪一抖，枪尖带着寒芒。
林冲（怒吼）：奸贼！纳命来！
△ 陆谦惊恐后退，林冲一枪刺穿差拨胸膛，鲜血喷溅在洁白的雪地上 → 富安想逃，被林冲回身一枪斩断双腿，倒在雪地哀嚎 → 陆谦跪地求饶，面容扭曲变形。
```

### 4.6 感官细节添加

**优秀的剧本需要调动多种感官：**

| 感官 | 描述词汇示例 |
|------|-------------|
| **视觉** | 颜色（灰白、暗红、鲜红）、光影（侧光、剪影、火光映照）、构图（对角线、中心对称） |
| **听觉** | 呼啸声、咯吱声、噼啪声、沉重、急促、凄厉 |
| **触觉** | 寒风刺骨、冰冷的雪、灼热的愤怒 |
| **情绪** | 孤寂、惊疑、压抑、爆发、决绝 |

## 第五步：15秒集数节奏控制

### 5.1 每集镜头数量分配

| 集数类型 | 镜头数 | 节奏 | 适用场景 |
|----------|--------|------|----------|
| 情绪/对白 | 3-4镜 | 4-5秒/镜 | 内心戏、对话、情感积累 |
| 动作/打斗 | 5-7镜 | 2-3秒/镜 | 战斗、追逐、高潮 |
| 蒙太奇/序列 | 6-8镜 | 2秒/镜 | 闪回、快速剪辑 |

### 5.2 情绪弧线

每集15秒必须有情绪起伏：

```
0-3秒：建立情绪/氛围
   ↓
3-9秒：情绪上升/冲突发展
   ↓
9-12秒：情绪高潮/关键揭示
   ↓
12-15秒：情绪释放/悬念收尾
```

### 5.3 集数衔接

**尾帧描述必须详细：**

```
✅ 正确：
**场景**：庭院中，一个孩童正追逐着跑向庭院角落的大水缸。
**画面**：
- 主体：5-6岁男童（浅褐色短衫），奔跑姿态
- 背景：大水缸（陶制，深褐色，水面平静）
- 光线：春日午后阳光，从右上角斜射
- 构图：孩童在画面左侧，水缸在右侧，形成对角线构图
- 氛围：看似平常却暗藏危机

**用途**：作为E02的开场，衔接意外发生的瞬间。

❌ 错误：
林冲走向山神庙。
```

## 第六步：视觉与听觉设计

### 6.1 色彩规划

| 阶段 | 主色调 | 情感目的 | 示例（林冲） |
|------|--------|----------|-------------|
| 开场 | 灰白、墨黑 | 孤寂压抑 | E1: 灰白墨黑，寒冷孤寂 |
| 发展 | 暖黄、灰蓝 | 危机暗藏 | E3: 暖黄火光与灰蓝外景对比 |
| 高潮 | 暗红、鲜红 | 愤怒/血腥 | E5: 鲜红鲜血与纯白雪地对比 |
| 结尾 | 苍茫白 | 余韵 | 天地苍茫，只剩脚印 |

### 6.2 视觉母题

选择1-3个视觉元素贯穿全片：

| 视觉母题 | 出现场景 | 象征意义 |
|----------|----------|----------|
| 风雪 | 全程 | 命运无情、环境压迫 |
| 长枪 | E1/E5 | 从看守武器到复仇利器 |
| 酒葫芦 | E1/E3/E5 | 逃避 vs 面对现实 |

### 6.3 音效设计

**每集音效包含三层：**

```
音效设计：
- 音乐：[配乐风格] + [乐器] + [节奏]
- 音效：[环境音] + [动作音] + [特殊音效]
- 对话：[对白内容]（如有）
```

**示例（林冲 E4）：**
```
音效设计：
- 音乐：低沉压抑的鼓点，逐渐加快，模拟心跳
- 音效：门外的风雪声、泼油声、火光燃烧声、林冲咬牙的咯咯声
- 对话：陆谦等人的阴谋对话（画外音）
```

```
第1集
1-1 日 外 草料场
道具：花枪、酒葫芦
出场人物：林冲

△ 【空镜】水墨风格渲染的沧州城外，漫天大雪如撒盐，天地间只剩黑白二色。
△ 远景俯拍，几间破败的草屋孤零零立在雪原中，四周是堆积如山的草料。
△ 镜头拉近，林冲头戴毡笠，身披破旧蓑衣，独自倚在草料堆旁。
△ 他抬起头，眼神中透着深深的疲惫与无奈，呼出的白气瞬间消散。
林冲（os）：以此赎罪，只盼能熬过这寒冬，再回东京……
△ 一阵狂风卷过，雪花猛地拍打在他脸上，林冲下意识地裹紧了蓑衣。
△ 他拿起腰间的酒葫芦晃了晃，却已是空空如也。
△ 狂风呼啸声骤然放大，仿佛野兽的低鸣，草料场的旗帜被吹得猎猎作响。
【字幕：沧州草料场 看守 林冲】
△ 林冲提着花枪，转身走向风雪深处，背影萧索。
```

## 快速检查清单

在完成剧本前，确认：

### 格式检查
- [ ] 每个镜头都以 `△ ` 开头
- [ ] 运镜使用了具体术语（推/拉/摇/移/跟/环绕/升降）
- [ ] 对白正确标注（os/vo/情绪）
- [ ] 包含感官细节（视觉+听觉+触觉）

### 内容检查
- [ ] 每集有明确的情绪弧线
- [ ] 集数之间衔接自然（尾帧→首帧）
- [ ] 人物性格一致且有发展
- [ ] 核心冲突得到解决
- [ ] 主题得到表达

### 技术检查
- [ ] 每集控制在15秒（3-7个镜头）
- [ ] 尾帧描述详细（场景+画面+光线+构图+氛围）
- [ ] 色彩规划明确
- [ ] 音效设计完整（音乐+音效+对话）
